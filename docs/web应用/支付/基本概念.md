# 基本概念

## 公众号

这是一个**统称**，通常指微信公众平台上注册的账号，包含以下两种主要类型：

### 订阅号

- **定位**：信息传播，适合媒体、个人、自媒体
- **推送频率**：每天可群发1次消息
- **消息展示**：折叠在"订阅号消息"文件夹中
- **功能权限**：基础接口，侧重内容推送
- **认证**：个人/企业均可注册，企业认证后功能更多

### 服务号

- **定位**：服务交互，适合企业、政府、组织
- **推送频率**：每月可群发4次消息
- **消息展示**：直接显示在聊天列表，有消息提醒
- **功能权限**：高级接口，支持**微信支付**、模板消息、卡券、客服等
- **认证**：必须企业/组织资质，需微信认证（300元/年）

## 对比

| 对比项               | 订阅号                      | 服务号                                |
| -------------------- | --------------------------- | ------------------------------------- |
| **定位**             | 信息传播、内容推送          | 服务交互、业务办理                    |
| **适用主体**         | 个人、媒体、企业            | 企业、政府、组织（不支持个人）        |
| **认证费用**         | 个人免费；企业认证 300元/年 | 微信认证 300元/年                     |
| **消息展示位置**     | 折叠在"订阅号消息"文件夹中  | 直接显示在聊天列表（类似好友）        |
| **消息提醒**         | ❌ 无红点/声音提醒           | ✅ 有红点+声音提醒                     |
| **群发次数**         | 每天 1 次                   | 每月 4 次                             |
| **微信支付**         | ❌ 不支持                    | ✅ 支持                                |
| **微信卡券**         | ❌ 不支持                    | ✅ 支持                                |
| **模板消息**         | ❌ 不支持                    | ✅ 支持（需用户触发或特定场景）        |
| **网页授权**         | ❌ 不支持获取用户基本信息    | ✅ 支持获取用户 OpenID、基本信息       |
| **生成带参数二维码** | ❌ 不支持                    | ✅ 支持（10万个临时码 + 10万个永久码） |
| **自定义菜单**       | 支持（无高级接口）          | 支持（含点击事件、跳转网页、扫码等）  |
| **高级接口**         | ❌ 大部分不支持              | ✅ 几乎全部支持                        |

## JS安全域名

JS 安全域名（JS 接口安全域名）是微信公众号调用微信 JS-SDK（如分享、支付、定位、卡券等）的前置必要条件。只有配置了安全域名的页面，才能正常调用微信开放的 JS 接口。

![image-20260527153802338](https://markdown-lai.oss-cn-hangzhou.aliyuncs.com/typora/image-20260527153802338.png)

| 步骤                | 操作                                                         | 说明                                   |
| ------------------- | ------------------------------------------------------------ | -------------------------------------- |
| **1. 下载验证文件** | 点击「设置」后，下载微信生成的 `MP_verify_xxx.txt` 文件      | 文件名随机生成，不要修改文件名和内容   |
| **2. 上传验证文件** | 将文件上传到域名 **根目录**（如 `https://yourdomain.com/MP_verify_xxx.txt`） | 必须能通过外网直接访问，不能放子目录   |
| **3. 填写域名**     | 在公众号后台输入域名，**不加协议头**（如 `yourdomain.com`）  | 只填域名，不写 `http://` 或 `https://` |
| **4. 保存验证**     | 点击保存，微信自动校验文件是否可访问                         | 验证通过后即生效                       |
