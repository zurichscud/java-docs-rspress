# 常量

## 全局常量

当你写 `public static final` 时，它属于**类级别**的常量。通常全大写命名，所有实例共享同一个值。

```java
public class AppConfig {
    // 全局常量：任何地方都能访问，且无法修改
    public static final String API_URL = "https://api.example.com";
    public static final int MAX_RETRY_COUNT = 3;
}

// 在其他类中使用：直接通过类名调用
String url = AppConfig.API_URL;
```





## 实例常量

如果不加 `static`，它就是**实例级别**的常量。每个对象都可以有不同的值，但**一旦在构造函数中确定，该对象运行期间就无法更改**。

```java
public class User {
    // 每个用户的ID在创建后不可更改，但不同用户的ID可以不同
    public final long id; 
    public String name;

    // 必须在构造函数中对 final 变量进行初始化
    public User(long id, String name) {
        this.id = id;
        this.name = name;
    }
}

// 在其他类中使用：
User user = new User(10086, "张三");
System.out.println(user.id); // 允许读取
// user.id = 10087;          // 编译报错！不允许修改
```
