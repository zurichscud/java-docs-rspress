# 单文件上传

## boundary

`boundary` 是 **multipart/form-data** 里一个非常关键的概念，用来把不同的表单部分（fields 或 files）**分隔开**。它本身不是你要传给服务器的数据，而是 HTTP 协议用来解析请求体的标识。

```http
### 单文件上传示例（boundary=aaa）
POST https://api.example.com/upload
Content-Type: multipart/form-data; boundary=aaa

--aaa
字段1
--aaa
字段2
--aaa--
```

在HTTP中boundary的格式：

- 开头：`--boundary`
- 分隔：`--boundary`
- 结束：`--boundary--`

字段的格式：

```http
Content-Disposition: form-data; name="字段名"

字段值
```

字段名通常由后端定义，例如创建了username字段和file字段：

```http
--aa
Content-Disposition: form-data; name="username"

zurichscud
--aa
Content-Disposition: form-data; name="file"; filename="example.txt"
Content-Type: text/plain

(文件内容)
--aa--
```

对于文件类型的字段还需要添加额外的配置：

- filename：文件在本地的名字
- Content-Type：文件的MIME类型

示例：

```http
### 单文件上传示例
POST https://api.example.com/upload
Content-Type: multipart/form-data; boundary=----WebKitFormBoundary7MA4YWxkTrZu0gW

------WebKitFormBoundary7MA4YWxkTrZu0gW
Content-Disposition: form-data; name="file"; filename="example.txt"
Content-Type: text/plain

< ./example.txt
------WebKitFormBoundary7MA4YWxkTrZu0gW--
```

::: tip

`< ./example.txt`是Rest Client识别的格式，真实发送的HTTP请求会替换为二进制数据

:::

## 实现

```java
package com.example.uploaddemo.controller;

import org.springframework.http.HttpStatus;
import org.springframework.http.ResponseEntity;
import org.springframework.web.bind.annotation.*;
import org.springframework.web.multipart.MultipartFile;

import java.io.File;
import java.io.IOException;

@RestController
@RequestMapping("/upload")
public class FileUploadController {

    // 单文件上传接口
    @PostMapping
    public ResponseEntity<String> uploadFile(
            @RequestParam("file") MultipartFile file,
            @RequestParam(value = "userId", required = false) String userId) {

        if (file.isEmpty()) {
            return ResponseEntity.status(HttpStatus.BAD_REQUEST).body("文件为空");
        }

        // 文件保存路径（可以改成你自己的路径）
        String uploadDir = System.getProperty("user.dir") + "/uploads/";
        File dir = new File(uploadDir);
        if (!dir.exists()) dir.mkdirs();

        try {
            File dest = new File(uploadDir + file.getOriginalFilename());
            file.transferTo(dest); // 保存文件
            return ResponseEntity.ok("上传成功，文件名：" + file.getOriginalFilename()
                    + ", userId=" + userId);
        } catch (IOException e) {
            e.printStackTrace();
            return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR)
                    .body("上传失败：" + e.getMessage());
        }
    }
}
```
