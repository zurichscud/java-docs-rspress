# 签名（SignName）

![image-20260528093156624](https://markdown-lai.oss-cn-hangzhou.aliyuncs.com/typora/image-20260528093156624.png)

## 基本概念

短信前面的：

```
【淘宝】
【京东】
【你的公司】
```

这个就是签名。

例如：

```
【法务系统】您的验证码是123456
```

这里的`法务系统`就是签名。

## 签名来源

-  企事业单位名

- 已注册商标名

- 已上线APP

::: warning

2026年4月27日：根据运营商的最新合规要求，阿里云短信服务将**不再支持"已上线APP"**作为签名来源。

:::

## 签名名称

### 企事业单位名

签名来源为“企事业单位名”的签名须为企业名称的全称或简称。

- 全称：全称需完全匹配企业或组织/机构证照名称。

- 简称：简称需包含在组织/机构证照的全称中，包含企业品牌名并且能唯一标识企业主体，企业简称不能为中性词或对应到多家企业，不能跳字、不能乱序、不能省略主要信息。

  示例：

  - 企业名称：杭州阿里云网络技术有限公司
  - 合规签名示例：【杭州阿里云网络技术】
  - 不合规签名示例：
    - 【阿里】：存在多个地区的阿里公司，无法体现独特性和唯一性；
    - 【阿里科技】：签名没有完全包含在企业名称全称中；
    - 【杭里网络】、【杭州阿里网】：签名没有包含企业品牌名，存在跳字。

### 已注册商标名

- 商标应在[国家知识产权局商标局-中国商标网](https://sbj.cnipa.gov.cn/sbj/index.html)中可查且已完成注册，商标注册主体和签名关联资质主体一致。

- 具备全网唯一性的商标： 商标名称无歧义的情况下，签名须与商标名一致，包含大小写。

- 不具备全网唯一性的商标： 若同一商标被多家企业注册，签名须补充企业信息以作区分。

  **说明**

  例如：A公司和B公司均注册了商标“XXX”，A公司申请商标类签名时，运营商侧可报备通过的签名内容可能为“XXX”或“AXXX”或“A公司XXX”，具体需以运营商实际审核结果为准。
