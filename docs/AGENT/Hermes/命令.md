# 命令

| 命令                  | 功能               |
| --------------------- | ------------------ |
| `/help`               | 显示所有可用命令   |
| `/tools`              | 列出可用工具       |
| `/model`              | 交互式切换模型     |
| `/personality pirate` | 尝试一个有趣的人格 |
| `/save`               | 保存对话           |

| 命令                | 说明                               |
| ------------------- | ---------------------------------- |
| `hermes`            | 开始聊天                           |
| `hermes model`      | 选择 LLM provider 和模型           |
| `hermes tools`      | 配置每个平台启用的工具             |
| `hermes setup`      | 完整配置向导（一次性配置所有内容） |
| `hermes doctor`     | 诊断问题                           |
| `hermes update`     | 更新到最新版本                     |
| `hermes gateway`    | 启动消息 gateway                   |
| `hermes --continue` | 恢复上次会话                       |
