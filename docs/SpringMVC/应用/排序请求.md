# 排序请求



## 单字段排序

单字段排序只需传递一个排序字段名和方向。

### 方案一：URL 参数

```java
GET /api/users?sortBy=createTime&order=desc
```

```js
axios.get('/api/users', {
  params: {
    sortBy: 'createTime',
    order: 'desc'
  }
});
```

### 方案二：JSON 参数

```http
POST /api/users/search
Content-Type: application/json

{
  "page": 1,
  "size": 10,
  "sortBy": "createTime",
  "order": "desc"
}
```

## 多字段排序

在前端处理多字段排序时，后端通常需要知道三个核心要素：**排序列名**、**排序顺序（asc/desc）**，以及多列之间的**优先级顺序**。

### 方案一：URL 参数

利用 HTTP 请求中同名参数可以传递多次的特性，或者直接用逗号分隔字段与方向。

顺序即代表优先级。先出现的先排序。

```java
GET /api/users?page=1&size=10&sort=status,desc&sort=createTime,asc
```

```js
axios.get('/api/users', {
  params: {
    sort: ['status,desc', 'createTime,asc']
  }
});
```

### 方案二：JSON 参数

适合复杂请求或 POST 导出

```http
POST /api/users/search
Content-Type: application/json

{
  "page": 1,
  "size": 10,
  "sorts": [
    { "field": "status", "order": "desc" },
    { "field": "createTime", "order": "asc" }
  ]
}
```
