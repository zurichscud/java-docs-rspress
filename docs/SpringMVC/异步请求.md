# 异步请求

异步请求（Asynchronous Request）是指前端在向后端发送请求时，**不需要等待服务器响应，就可以继续执行后面的代码或进行页面操作**。

| **特征**     | **同步请求**                                                 | **异步请求**                                                 |
| ------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **页面状态** | **卡死/白屏**。等待期间用户无法点击任何按钮，页面处于冻结状态。 | **完全流畅**。用户可以继续滚动页面、点击其他按钮，不受任何影响。 |
| **数据更新** | 必须**刷新整个页面**才能看到新数据。                         | **局部刷新**。只有需要变动的那个小区域被更新。               |
| **典型技术** | 早期的 HTML `<form>` 表单提交、直接修改 `window.location.href`。 | `Ajax`、`Fetch API`、`Axios`（前端常用）。                   |
