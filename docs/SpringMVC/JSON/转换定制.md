# 转换定制

## 兼容LocalDateTime

Json默认只支持Date类型的日期转换，但现在大部分使用的是LocalDate、LocalDateTime、LocalTime。因此你需要添加JsonDeserializer针对LocalDateTime、LocalDate、LocalTime。

```java
@Configuration
public class JacksonConfig {

    private static final String DEFAULT_DATE_TIME_FORMAT = "yyyy-MM-dd HH:mm:ss";
    private static final String DEFAULT_DATE_FORMAT = "yyyy-MM-dd";
    private static final String DEFAULT_TIME_FORMAT = "HH:mm:ss";

    @Bean
    public Jackson2ObjectMapperBuilderCustomizer jackson2ObjectMapperBuilderCustomizer() {
        DateTimeFormatter dateTimeFormatter = DateTimeFormatter.ofPattern(DEFAULT_DATE_TIME_FORMAT);
        DateTimeFormatter dateFormatter = DateTimeFormatter.ofPattern(DEFAULT_DATE_FORMAT);
        DateTimeFormatter timeFormatter = DateTimeFormatter.ofPattern(DEFAULT_TIME_FORMAT);

        return builder -> builder
                .timeZone(TimeZone.getTimeZone("GMT+8"))
                .serializerByType(LocalDateTime.class, new LocalDateTimeSerializer(dateTimeFormatter))
                .serializerByType(LocalDate.class, new LocalDateSerializer(dateFormatter))
                .serializerByType(LocalTime.class, new LocalTimeSerializer(timeFormatter))
                .deserializerByType(LocalDateTime.class, new LocalDateTimeDeserializer(dateTimeFormatter))
                .deserializerByType(LocalDate.class, new LocalDateDeserializer(dateFormatter))
                .deserializerByType(LocalTime.class, new LocalTimeDeserializer(timeFormatter));
    }
}

```

`LocalDateTimeSerializer`、`LocalDateSerializer` 等，全都是 Jackson 官方为了适配 Java 8 日期新特性而为你准备好的**现成工具类**。



## 自定义JsonDeserializer

如果你有特殊的类型转换，例如将1520*1080 字符串转为`Size`

```java
public class Size {
    private Integer width;
    private Integer height;
    // Getter, Setter, 构造函数...
}
```

设计序列化器（serializer）和反序列化（deserializer）器

| 方法                 | 作用                   | 方向            |
| -------------------- | ---------------------- | --------------- |
| `serializerByType`   | 根据类型指定序列化器   | Java对象 → JSON |
| `deserializerByType` | 根据类型指定反序列化器 | JSON → Java对象 |

### JsonDeserializer

```java
import com.fasterxml.jackson.core.JsonParser;
import com.fasterxml.jackson.databind.DeserializationContext;
import com.fasterxml.jackson.databind.JsonDeserializer;

import java.io.IOException;


public class SizeDeserializer 
        extends JsonDeserializer<Size> {


    @Override
    public Size deserialize(
            JsonParser p,
            DeserializationContext ctxt
    ) throws IOException {

        String text = p.getText();

        String[] arr = text.split("\\*");

        if (arr.length != 2) {
            throw new IllegalArgumentException(
                    "Size格式错误：" + text
            );
        }

        int width = Integer.parseInt(arr[0]);
        int height = Integer.parseInt(arr[1]);

        return new Size(width, height);
    }
}
```

### JsonSerializer

```java
import com.fasterxml.jackson.core.JsonGenerator;
import com.fasterxml.jackson.databind.JsonSerializer;
import com.fasterxml.jackson.databind.SerializerProvider;

import java.io.IOException;


public class SizeSerializer extends JsonSerializer<Size> {

    @Override
    public void serialize(
            Size value,
            JsonGenerator gen,
            SerializerProvider serializers
    ) throws IOException {

        String result = value.getWidth()
                + "*"
                + value.getHeight();

        gen.writeString(result);
    }
}
```

### 注册

在JacksonConfig的Builder中进行注册：

```java
@Configuration
public class JacksonConfig {

    @Bean
    public Jackson2ObjectMapperBuilderCustomizer jacksonCustomizer() {

        return builder -> builder
                .serializerByType(
                        Size.class,
                        new SizeSerializer()
                )
                .deserializerByType(
                        Size.class,
                        new SizeDeserializer()
                );
    }
}
```
