# 端口

## 常见端口

| 端口  | 占用服务 | 说明                   |
| ----- | -------- | ---------------------- |
| 22    | SSH      | Linux 远程登录         |
| 23    | Telnet   | 明文远程协议（不推荐） |
| 53    | DNS      | 域名解析               |
| 67/68 | DHCP     | 自动分配 IP            |
| 123   | NTP      | 时间同步               |

## Web / 应用服务

| 端口      | 占用服务             | 说明              |
| --------- | -------------------- | ----------------- |
| 80        | HTTP                 | Web 默认端口      |
| 443       | HTTPS                | 加密 Web          |
| 8080      | Tomcat / Spring Boot | 常见 Java 后端    |
| 3000      | Node.js / React      | 前端开发常用      |
| 5173      | Vite                 | Vue3 默认开发端口 |
| 8000/8001 | 自定义服务           | 开发环境常用备用  |

## 数据库

| 端口  | 占用服务   | 说明               |
| ----- | ---------- | ------------------ |
| 3306  | MySQL      | 最常见关系型数据库 |
| 5432  | PostgreSQL | 另一种主流数据库   |
| 1521  | Oracle     | 企业级数据库       |
| 1433  | SQL Server | 微软数据库         |
| 6379  | Redis      | 缓存数据库         |
| 27017 | MongoDB    | 文档型数据库       |



## 中间件

| 端口  | 占用服务    | 说明             |
| ----- | ----------- | ---------------- |
| 9092  | Kafka       | 分布式消息队列   |
| 5672  | RabbitMQ    | 消息队列协议端口 |
| 15672 | RabbitMQ UI | 管理后台         |
