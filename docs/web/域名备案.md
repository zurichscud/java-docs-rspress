# 域名备案

如果你买的是中国大陆服务器，并且网站要在中国大陆访问，那么域名通常需要备案。

## 域名备案的几种情况

| 服务器所在地                         | 是否需要备案 |
| ------------------------------------ | ------------ |
| 中国大陆（阿里云北京、上海、深圳等） | 需要备案     |
| 中国香港                             | 不需要备案   |
| 日本、新加坡、美国等海外服务器       | 不需要备案   |

## 备案流程

1. 域名实名认证
2. 购买中国大陆云服务器
3. 在云服务商备案系统提交资料
4. 工信部审核
5. 获得ICP备案号
6. 网站底部展示备案号

## 备案结果

备案号通常长这样

```
粤ICP备2026123456号
```

## 其他解释

ICP备案是针对**主域名**备案的，例如：

```
example.com
```

备案通过后，以下域名都可以正常解析和访问：

```
example.com
www.example.com
a.example.com
b.example.com
api.example.com
static.example.com
```
