# 响应

## 响应头分类

响应头是给客户端（浏览器）阅读的

### CORS

- **`Access-Control-Allow-Origin`**：允许哪些网站跨域访问。

示例：`Access-Control-Allow-Origin: http://localhost:8080`（或者 `*`，代表允许所有人，但允许所有人时不能带 Cookie）。

- **`Access-Control-Allow-Methods`**：允许前端使用哪些 HTTP 方法。

示例：`Access-Control-Allow-Methods: GET, POST, PUT, DELETE`。

- **`Access-Control-Allow-Headers`**：在预检请求（OPTIONS）中，告知前端允许携带哪些自定义请求头（比如 `Authorization`）。

- **`Access-Control-Allow-Credentials`**：是否允许前端跨域发送 Cookie。如果设为 `true`，前端 Axios 必须配置 `withCredentials: true`。

### 响应体描述

这些头让浏览器知道它收到的到底是个什么东西，该怎么渲染：

**`Content-Type`**：告知返回数据的媒体类型（MIME type）。

- 示例：`Content-Type: application/json;charset=UTF-8`（告诉 Vue 这是一个 JSON 串，浏览器会自动帮你把它转成 JS 对象）。
- 示例：`Content-Type: image/png`（告诉浏览器这是一张图片，直接渲染出来）。

**`Content-Length`**：响应体的数据大小（字节数）。浏览器用来显示下载进度条或判断数据是否接收完整。

**`Content-Disposition`**：控制文件是直接在浏览器里打开，还是作为附件下载。

- 示例：`Content-Disposition: attachment; filename="report.xlsx"`（Java 导出 Excel 时必备，浏览器看到 `attachment` 就会直接弹出下载框）。

### 会话与安全

**`Set-Cookie`**：**极其重要**。这是后端向浏览器写入 Cookie 的唯一方式。

- 示例：`Set-Cookie: JSESSIONID=abc123g; Secure; HttpOnly; SameSite=Strict`
- 浏览器收到后会把这个 Cookie 存起来，以后每次同源请求都会自动在请求头里带上它。
- `HttpOnly` 属性非常重要，它能防止黑客通过 JS 脚本（XSS 攻击）偷走 Cookie。

**`Strict-Transport-Security` (HSTS)**：告诉浏览器：“以后访问我这个网站，必须强制使用 HTTPS，就算用户敲了 `http://` 也要自动变成 `https://`。”

### 缓存

为了让 Vue 页面加载更快，减少对 Java 后端的轰炸，后端可以通过响应头控制浏览器的缓存策略：

- **`Cache-Control`**：缓存控制大总管。
  - 示例：`Cache-Control: max-age=3600`（告诉浏览器：“这个资源一小时内都是新鲜的，直接用本地缓存，别来烦我（后端）。”）
  - 示例：`Cache-Control: no-cache`（告诉浏览器：“每次使用前，你必须先来问问我数据变了没。”）
- **`ETag`**：资源的“指纹”（版本标识）。
  - 下次前端再请求时，会在请求头带上 `If-None-Match: <ETag>`。Java 后端发现指纹没变，就会直接返回 **304 Not Modified** 响应状态码（而且响应体为空，极省流量）。

## 自定义响应头

服务器可以在响应头中添加自定义响应头，我们还需要设置`Access-Control-Expose-Headers`，浏览器才能够读取到我们自定义响应头的值

```js
app.use((req,res,next)=>{
  res.set('Lai-Header', 'lai')
  res.set('Access-Control-Expose-Headers', 'Lai-Header')
  next()
})
```

```js
fetch('http://localhost:3000/get')
.then(res => {
    // 获取 Lai-Header响应头
    const laiHeader = res.headers.get('Lai-Header');
    console.log('Lai-Header:', laiHeader);

    // 获取响应体数据
    return res.text();
})
.then(data => console.log('Response data:', data))
.catch(err => console.log('Error:', err));

```
