# 文件树示例

下面是一个使用 `tree` 代码块渲染的文件树：

```tree
my-project/
├── src/
│   ├── components/
│   │   ├── Button.tsx
│   │   └── Header.tsx
│   ├── utils/
│   │   └── helpers.ts
│   └── index.ts
├── public/
│   └── favicon.ico
├── package.json
├── tsconfig.json
└── README.md
```
