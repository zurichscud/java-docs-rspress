# 介绍

Rspress 是一个基于 [Rsbuild](https://rsbuild.rs/) 的静态站点生成器，使用 React 框架进行渲染。它默认提供文档主题，你可以使用 Rspress 快速搭建一个文档站点。

## 为什么选择 Rspress

- **构建性能**。核心编译模块基于 Rust 前端工具链，提供毫秒级启动和更极致的开发体验。
- **AI 原生**。技术文档不仅服务于人类读者，还可以通过 SSG-MD 让 AI 更好地理解和利用。
- **MDX 支持**。MDX 是一种强大的内容编写方式，允许你在 Markdown 中使用 React 组件。
- **基础能力**。包括全文搜索、国际化、多版本支持、组件库文档等。
- **可扩展性**。提供内置的插件系统，支持通过插件 API 扩展 Rspress。
