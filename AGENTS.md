## Project Overview

Rspress v2 documentation site with custom theme, covering Java/Spring/Spring Boot/web development notes. Content is in Chinese with some English technical terms.

## Commands

```bash
npm run dev          # Start dev server
npm run build        # Build for production
npm run preview      # Preview production build
npm run gen:recent-updates  # Regenerate recent-updates.json and stats.json from git history
```

## Architecture

- **`docs/`** — Documentation content (MD/MDX). Each top-level folder is a module (guide, JAVA, Springboot, web, Spring, 数据库, Maven, api)
- **`theme/`** — Custom theme extending `@rspress/core/theme-original`
  - `components/HomeLayout/index.tsx` — Custom homepage with module cards and recent updates panel
  - `data/recent-updates.json` / `stats.json` — Generated by `scripts/generate-recent-updates.mjs` using git log
  - `index.css` — Global theme styles
- **`rspress.config.ts`** — Site config with remark plugin for normalizing code block language names (e.g., `SQL` → `sql`)
- **`scripts/generate-recent-updates.mjs`** — Scans `docs/` for MD/MDX files, extracts titles from frontmatter or first `#` heading, uses `git log` for timestamps

## Sidebar Navigation

Defined per directory via `_meta.json` files. Supports `dir-section-header`, `dir`, and `file` types. See existing `_meta.json` files for format examples.

## Plugins

- `rspress-plugin-file-tree` — File tree component
- `rspress-plugin-mermaid` — Mermaid diagram rendering
